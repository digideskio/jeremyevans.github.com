OpenBSD

I'm an OpenBSD developer (jeremy@), responsible for OpenBSD's ruby ports.

I'm the maintainer of the following OpenBSD ports:

  • audio/aqualung: advanced music player
  • comms/c3270: curses-based IBM 3270 terminal emulator
  • databases/ruby-ldap: Ruby interface to some common LDAP libraries
  • databases/ruby-pg: Ruby interface to PostgreSQL RDBMS
  • databases/ruby-sequel: The database toolkit for ruby
  • editors/scintilla: source code editing component for GTK+
  • editors/scite: flexible and small GTK+ editor
  • multimedia/subtitleripper: DVD subtitle ripper
  • textproc/hyperestraier: high performance full-text search system
  • textproc/ruby-hyperestraier: Ruby interface to hyperestraier
